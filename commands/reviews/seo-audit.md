# SEO Fundamentals Audit

Perform a comprehensive check of SEO basics for this project.

## Meta Tags and Structure

1. **Title Tags**:
   - Are unique titles set for each page?
   - Are they descriptive and within character limits (50-60 chars)?

2. **Meta Descriptions**:
   - Does each page have a unique meta description?
   - Are they compelling and within limits (150-160 chars)?

3. **Heading Hierarchy**:
   - Is there exactly one H1 per page?
   - Does heading structure follow logical hierarchy (H1 > H2 > H3)?

4. **Canonical URLs**:
   - Are canonical tags properly set?
   - Do they point to the correct URLs?

## Technical SEO

1. **Sitemap**:
   - Does `sitemap.xml` exist and is it accessible?
   - Is it registered in `robots.txt`?
   - Does it include all important pages?

2. **Robots.txt**:
   - Does `robots.txt` exist?
   - Are there any unintended blocks?
   - Is the sitemap referenced?

3. **Structured Data**:
   - Is JSON-LD schema implemented?
   - Is it valid and appropriate for content type?

4. **Image Optimization**:
   - Do images have alt text?
   - Are images appropriately sized and compressed?

## Performance Factors

1. **Core Web Vitals** indicators in the code:
   - Lazy loading implementation
   - Critical CSS handling
   - Font loading strategy

2. **Mobile-Friendliness**:
   - Responsive viewport meta tag
   - Touch-friendly elements

## Output Format

1. **Critical Issues**: Must-fix problems affecting indexing
2. **Warnings**: Important improvements for rankings
3. **Recommendations**: Nice-to-haves for optimization
4. **Checklist**: Simple pass/fail for each element checked
